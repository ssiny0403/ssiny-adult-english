## 프로젝트 개요

**목적:** 성인 수강생 대상 랜딩 페이지 (전화 문의 전환 유도)
**파일 형식:** 단일 HTML 파일 (GitHub Pages 배포용)
**언어:** 한국어 (UI 및 카피 전체)
**기술 스택:** HTML + CSS + Vanilla JS (외부 라이브러리 없음, 단일 파일 완결)

## 타겟 페르소나

### 주요 대상
- **주부 (30–40대 육아 중인 엄마):** 아이들 교육에 관심 많고, 자기 개발에도 목마른 분들. 오랫동안 영어를 놓았지만 다시 시작하고 싶음.
- **시니어 (50대 이상):** 여행, 일상 회화가 목표. 느리지 않게, 자신감 있게 말하고 싶음.
- **직장인 / 프리랜서:** 업무 외 시간에 조용하고 편안한 환경에서 영어를 배우고 싶음.

### 공통 심리
- "오랫동안 배웠는데 말을 못한다"는 좌절감이 있음
- 학원 같은 딱딱한 분위기는 싫음
- 선생님을 신뢰할 수 있어야 등록을 결심함
- **"써먹는 영어, 말문이 트이는 영어, 문장으로 말하는 여행지 영어"** — 이게 핵심 욕구

## 페이지 구조 (섹션 순서)

### 1. Hero 섹션
- **헤드라인:** "눈으로만 읽고 정작 써먹지 못하는 영어는 그만, 말문이 트이는 영어" (메인 카피)
- **서브카피:** 카페에서 커피 한 잔 마시며 나누는 영어 대화 — 씬잉글리쉬 스튜디오 성인반
- **CTA 버튼:** "지금 전화 상담하기" → `tel:` 링크 (클릭 즉시 전화 연결)
- 배경: 우드톤 기반, 따뜻하고 여유로운 이미지 분위기 (CSS로 표현)

### 2. 공감 섹션 — "혹시 이런 경험 있으신가요?"
- 타겟 페르소나의 좌절 포인트를 짧고 직관적으로 나열
  - "여행지에서 아는 단어인데 내 발음을 못 알아듣거나, 상대 발음을 못 알아들어서 겁먹은 적 있으세요?"
  - "수년째 영어 공부를 해도 입이 안 떨어져요"
  - "외국인 앞에만 서면 머릿속이 하얘져요"
  - "학원은 너무 딱딱하고 부담스러워요"
- 전환 멘트: "씬잉글리쉬 성인반에서는 달라요"

### 3. 수업 방식 소개 — "커피챗 영어란?"

- 수업 주제: 일상 대화, 여행 영어, 자유 프리토킹
- 키워드 카드 형식으로 시각화:
  - ☕ 커피챗 분위기
  - 🗣️ 실제 말하기 중심
  - 👤 1:1 또는 소그룹
  - 📍 부산 교대역 5번 출구 (연제구)

### 4. 수강생 후기 / 변화 스토리

  - "6개월 후 해외 여행에서 처음으로 혼자 주문했어요" — 40대 주부

### 레이아웃
- **최대 너비:** `900px` (중앙 정렬, 좌우 패딩 `1.5rem`)
- **섹션 패딩:** `5rem 0` (데스크탑) / `3rem 0` (모바일)
- **카드 그리드:** CSS Grid, `repeat(auto-fit, minmax(260px, 1fr))`
- **모바일 우선** 반응형 (breakpoint: 768px)

### 시그니처 디자인 요소
> **"커피 링 (coffee ring) 텍스처 구분선"**
> 
> 섹션 간 구분에 CSS로 그린 원형 커피 링 모양 divider 사용.
> 직선 `<hr>` 대신, 옅은 우드 브라운 원호(arc) 또는 점선 원 형태.
> 학원 랜딩 페이지 어디에서도 보기 힘든 요소 — 커피챗 컨셉을 시각적으로 일관되게 유지.

## CTA 스펙

```html
<!-- 전화 연결 버튼 -->
<a href="tel:010-8070-4722" class="cta-button">
  📞 지금 전화 상담하기
</a>
```
- 색상: `--color-accent` (#92308D)
- 형태: 둥근 모서리 (`border-radius: 50px`), 여유있는 패딩
- hover: 살짝 위로 올라오는 `transform: translateY(-2px)` + 그림자
- **모바일에서 화면 하단 고정 플로팅 버튼** 추가 (sticky CTA)

## 애니메이션 지침

- 스크롤 진입 시 섹션 페이드인 (Intersection Observer, JS)
- 카드 hover 시 가볍게 위로 lift (`transform: translateY(-4px)`)
- **과도한 애니메이션 금지** — 성인 타겟, 신뢰감이 우선
- `@media (prefers-reduced-motion: reduce)` 반드시 처리
